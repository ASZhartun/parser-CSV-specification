# parser-CSV-specification

## Парсер спецификации железобетонных элементов

<h3>Цель</h3>
Формирует спецификацию и ведомость по КЖ на базе таблицы спецификации, в которой заполнены графы "Наименование" и
"Количество".</br>
Исходником является csv файл, получаемый в ходе экспорта таблицы автокада в csv файл.
<h3>Результат работы</h3>
Заполненная ведомость, дозаполненная спецификация.

<h3>Требования к начальной таблице</h3>
Таблица имеет:
<ul>
<li>заголовок</li>
<li>шапка колонок</li>
<li>правильное написание стандартных разделов (<i>Материалы, Сборочные единицы, Изделия закладные, Детали</i>)</li>
</ul>

<h3>Особенности</h3>
Парсер не учитывает пользовательские арматурные сетки, закладные изделия, каркасы.

Парсер учитывает <b>только</b> <i>арматурные стержни, арматурные сетки по ГОСТ</i>, бетон по СТБ.

<h2>Как запустить</h2>
<ol>
<li>выполнить package</li>
<li>поместить table.csv в каталог с package.jar</li>
<li>в cmd выполнить команду <code>java -jar <i>package</i>.jar "абсолютный путь к \table.csv"</code>. <i>Напр.</i>, 
   <code>java 
   -jar 
   parser-CSV-specification.jar "D:\parser\table.csv"</code></li>
</ol>
