# parser-CSV-specification
## Парсер спецификации железобетонных элементов
###Цель
Формирует спецификацию и ведомость по КЖ на базе таблицы спецификации, в которой заполнены графы "Наименование" и 
"Количество".\
Исходником является csv файл, получаемый в ходе экспорта таблицы автокада в csv файл.
###Результат работы: 
Заполненная ведомость, дозаполненная спецификация.

###Требования к начальной таблице
Таблица имеет:
- заголовок
- шапка колонок
- правильное написание стандартных разделов (*Материалы*, *Сборочные единицы*, *Изделия закладные*, *Детали*)

###Особенности
Парсер не учитывает пользовательские арматурные сетки, закладные изделия, каркасы.

Парсер учитывает **только** *арматурные стержни*, арматурные *сетки по ГОСТ*, бетон по СТБ.

###Как запустить
1. выполнить package
2. поместить table.csv в каталог с *package*.jar
3. в cmd выполнить команду <code>java -jar *package*.jar "абсолютный путь к \table.csv"</code>. *Напр.*, <code>java 
   -jar 
   parser-CSV-specification.jar "D:\parser\table.csv"</code>
